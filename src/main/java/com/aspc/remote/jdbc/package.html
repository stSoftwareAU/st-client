A JDBC driver to allow remote applications to interface with stSoftware.

To provide the facility within your Java application to interface with stSoftware
include stRemote.jar in your classpath and follow the following steps.
try
{
Class.forName("com.aspc.remote.jdbc.SoapDriver" );
}
catch (Exception e)
{
System.out.println("ERROR: failed to load JDBC driver.");
e.printStackTrace();
return;
}

Properties props = new Properties();
props.put( "user", id );
props.put( "password", password );

Connection connection = DriverManager.getConnection( "SOAP:192.168.7.1/my_data", props );
